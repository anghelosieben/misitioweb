iniciando un proyecto en github

github es un repositorio donde los programadores y otros pueden subir sus proyectos

otra ventaja de github es el poder tener un sitio, que es denominado github pages

en primer lugar debes crear un acuenta en github, es facil

tambien crear el repositorio con un nombre ej: misitioweb

luego instalar git, y desde el programa git (debe ser instalado en el equipo local), ejecutar comando, la linea de comandos es similares a los de linux puede usar pwd, cd, mkdir, etc...

tenemos que crear la carpeta en el equipo de manera local ej misitioweb

desde la terminal de comandos ejecutar

      $ git init
      $ git status
      $ git remote add origin "el link del proyeecto github"

ahora vamos a subir un archivo del equipo local al repositorio de git 

      $ git add index.html
      $ git commit -m "subiendo archivos"
      $ git push origin master
